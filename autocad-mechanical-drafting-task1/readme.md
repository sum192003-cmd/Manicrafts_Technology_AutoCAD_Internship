# Task 1 - 2D CAD Drawing

## Project Overview
This repository contains a 2D CAD drawing created using AutoCAD. The project includes the original DWG source file, PDF exports, and a screenshot preview for easy viewing.

## Files Included

| File Name | Description |
|------------|-------------|
| task 1.dwg | Original AutoCAD drawing file |
| task 1-Model_page-0001.jpg | Preview image of the drawing |
| task 1-Model.pdf | Model layout PDF export |
| screenshot task 1.jpg | layer manager image of the drawing |

## Software Used
- AutoCAD
- PDF Viewer
